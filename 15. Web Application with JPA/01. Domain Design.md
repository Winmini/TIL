# Domain Design



#### 도메인 모델 

![도메인 모델](./jpg/domain model.png)

회원은 여러 상품을 주문할 수 있다. 그리고 한 번 주문할 때 여러 상품을 선택할 수 있으므로 주문과 상품은 다대다 관계다. 상품은 여러개 등록될  수 있으므로 상품과 관계는 다대일 관계 등등이 된다.
